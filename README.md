# Movie and TV Show Database Frontend

Project Video : https://youtu.be/KrzfjzklWDo

Backend Repository: https://github.com/Harinder05/Movie_Database_API

This is the frontend application for managing movies and TV shows, built with React, Axios, and JavaScript.

## Prerequisites
To run this project locally, you need to have the following installed on your system:

- Node.js
- npm

## Installation
1. Clone the repository: `git clone https://github.com/Harinder05/movie_api_frontend.git`
2. Change to the project directory: `cd movie_api_frontend`
3. Install the dependencies: `npm install`

## Running the Frontend
To start the frontend application, run: `npm start`.
The application will be accessible at: `http://localhost:3000`

## Features
- User registration and sign-in
- Display a list of movies
- Add a new movie
- Update an existing movie
- Delete a movie
